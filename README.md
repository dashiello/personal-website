# personal-website
The site is live at [DashiellPepperman.com](https://dashiellpepperman.com/)

Personal website to be used as a resume.

To Run Locally:
1. Download from the branch you want
2. Open a terminal in the location you downloaded & type `yarn` to install dependencies (if you don't have yarn, it can be installed through npm)
3. run through live server by typing `.\node_modules\.bin\live-server`
